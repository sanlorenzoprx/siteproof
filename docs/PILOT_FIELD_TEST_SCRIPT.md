# SiteProof Pilot Field Test Script

## Test 1 — First launch

1. Open SiteProof while online.
2. Complete onboarding.
3. Open Pilot screen.
4. Confirm readiness score is visible.

Pass if the app loads and readiness checks appear.

## Test 2 — Create job

1. Tap Start Job.
2. Create a generator install job.
3. Confirm workflow stages appear.

Pass if stage checklist is generated from the template.

## Test 3 — Capture proof

1. Open a missing proof item.
2. Take a photo.
3. Confirm proof progress updates.
4. Open gallery/timeline.

Pass if the photo appears in the job, timeline, and required proof checklist.

## Test 4 — Voice note

1. Record a voice note.
2. Add a note that mentions a material or issue.
3. Confirm summary/extractions appear.

Pass if material/issue/change-order fields are populated or the raw note is saved offline.

## Test 5 — Export

1. Generate customer report.
2. Try inspector report with missing proof.
3. Confirm inspector export blocks or warns correctly.

Pass if customer report generates and inspector report respects readiness rules.

## Test 6 — Offline survival

1. Turn on airplane mode.
2. Close/reopen the app.
3. Open the same job.
4. Capture another note/photo.

Pass if the app remains usable and sync queue shows pending operations.
