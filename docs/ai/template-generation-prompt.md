# SiteProof Template Generation Prompt v1

Draft a SiteProof workflow template for the specified specialty or job type.

SiteProof templates should reduce missed documentation, speed field capture, and improve export packets.

Use canonical workflow stages:

1. Intake
2. Site Arrival
3. Existing Conditions
4. Active Work
5. Inspection Readiness
6. Completion
7. Customer Signoff
8. Export / Archive

For each stage include:

- required proof objects
- suggested photos
- suggested voice notes
- required metadata
- inspection relevance
- export section placement
- missing-proof warnings
- optional AI extraction fields
- field simplicity notes

Rules:

- Do not create legal/compliance certainty unless verified by human review.
- Prefer required proof sets over long forms.
- Minimize typing.
- Favor guided capture.
- Keep field screens simple.
- Use canonical primitives.

Return:

- template name
- specialty/job type
- workflow stages
- required proof list
- optional proof list
- export packet mapping
- missing-proof validation rules
- human review requirements
- assumptions and cautions
