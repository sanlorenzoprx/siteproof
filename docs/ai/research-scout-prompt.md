# SiteProof Research Scout Prompt v1

Research current information that may improve SiteProof strategy, templates, exports, field usability, or technical reliability.

Focus areas:

- contractor software complaints
- field documentation pain points
- Procore / ServiceTitan / Jobber / CompanyCam weaknesses
- generator installation documentation requirements
- electrical inspection documentation expectations
- Puerto Rico infrastructure and utility field realities
- Florida emergency power and storm-response needs
- insurance claim documentation expectations
- Android offline-first constraints
- Android background/battery restrictions
- field service photo documentation best practices

Return:

1. key findings
2. source links
3. implications for SiteProof
4. feature/template opportunities
5. risks or constraints
6. immediate low-effort improvements
7. governance updates recommended

Important:

- Prefer official, primary, or high-trust sources.
- Distinguish confirmed requirements from inferred best practices.
- Do not convert research into compliance claims without human validation.
