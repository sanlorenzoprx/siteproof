# SiteProof Export Quality Review Prompt v1

Review this SiteProof export, report, or packet structure.

For contractors, the export is the product. It must be clear, professional, defensible, and useful to customers, inspectors, insurance adjusters, and internal teams.

Evaluate:

- customer readability
- inspector usefulness
- proof ordering
- timeline clarity
- summary quality
- missing proof visibility
- photo labeling
- voice note summarization
- manifest inclusion
- hash/proof integrity information
- branding/professional appearance
- legal defensibility without overclaiming
- unnecessary clutter

Look for:

- vague summaries
- unorganized photos
- missing timestamps/GPS
- missing proof manifest
- weak job narrative
- duplicated sections
- excessive technical language for customer report
- insufficient technical detail for inspector report
- no clear completion state

Return:

1. export quality score
2. missing proof or weak evidence
3. ordering improvements
4. summary improvements
5. packet type recommendations
6. customer-facing clarity improvements
7. inspector-facing clarity improvements
8. legal/defensibility cautions
9. required export tests
