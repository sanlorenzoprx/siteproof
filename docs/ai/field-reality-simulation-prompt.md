# SiteProof Field Reality Simulation Prompt v1

Simulate real field users trying to use this SiteProof feature.

Personas to consider:

- generator installer during outage response
- Puerto Rico storm-response contractor
- Florida emergency power contractor
- electrical foreman
- exhausted technician at the end of the day
- new hire on first week
- inspector reviewing proof
- office admin preparing closeout packet

Simulate hostile conditions:

- no signal
- intermittent LTE
- low battery
- battery saver enabled
- bright sunlight
- gloves
- wet screen
- noisy environment
- ladder usage
- truck cab usage
- app backgrounded mid-capture
- Android process killed
- device reboot
- GPS unavailable
- nearly full storage

Evaluate:

1. where the user hesitates
2. where proof could be lost
3. where sync confidence is unclear
4. where typing becomes painful
5. where the user leaves capture momentum
6. where failure recovery is weak
7. what a stressed user would misunderstand
8. what should be automated, defaulted, or removed

Return:

- top field failure risks
- simplification recommendations
- durability recommendations
- telemetry to capture during pilot
- manual field test script
- feature decision: approve / revise / reject
